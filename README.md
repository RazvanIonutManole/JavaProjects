A collection of Java projects developed during my introductory computer science semester. This repository tracks my progression from basic syntax to building functional desktop applications integrated with AWS services.
The ui is in romanian and most of the projects are very simple since i made them while i was learning java for the first time.
Some more notable projects are in the "exemple" tab: a paint clone, a notepad clone and a calculator.

To run the project, make sure you have Java 17 installed and double click the "start.bat" file.
